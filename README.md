# Campaign App (Kotlin)

This is a Kotlin-based project for campaign management.

## Prerequisites

- Java 21 or later
- Gradle (wrapper included)

## Building

```bash
./gradlew build
```

## Running

```bash
./gradlew run
```

## Testing

```bash
./gradlew test
```

## Project Structure

- `src/main/kotlin/com/example/` - Main source code
- `src/test/kotlin/com/example/` - Unit tests
- `build.gradle.kts` - Build configuration
- `settings.gradle.kts` - Project settings
